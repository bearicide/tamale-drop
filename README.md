# Tamale Drop

Static GitHub Pages site. No build step, no PWA, no app-store cosplay.

## Edit these first
- `index.html`
  - Phone number in the `sms:` link
  - Facebook/Messenger link
  - Flavors and prices
  - Pickup city, address/window, and batch timing

## Publish
1. Create repo: `tamale-drop`
2. Upload these files
3. GitHub repo settings → Pages → deploy from main branch
4. Share the GitHub Pages URL

## Files
- `index.html`
- `style.css`
- `script.js`
- `assets/tamale-hero.svg`
